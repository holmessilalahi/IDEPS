IDEPS (Indonesian Earthquake Prediction System) is a web-based platform developed to predict earthquake magnitude and depth across Indonesia using historical seismic data.

This system was developed by Holmes Heriyanto Silalahi and leverages approximately 50 years of earthquake records obtained from the United States Geological Survey (USGS). By utilizing machine learning techniques, IDEPS aims to provide data-driven insights based on geographical inputs, particularly longitude and latitude coordinates.

The platform is designed to support research and exploration in earthquake pattern analysis, offering an accessible interface for users to simulate predictions based on past seismic activity.

Key Features:

Prediction of earthquake magnitude and depth
Utilization of long-term historical seismic datasets in Indonesia
Web-based interface for easy access and interaction
Machine learning-based prediction model
